# CSS Grid Standard Web Layout

This project demonstrates a standard website layout using CSS Grid.

## Layout Structure
- Header  
- Sidebar  
- Main Content  
- Footer  

## Concepts Practiced
- display: grid  
- grid-template-columns  
- grid-template-areas  
- Grid positioning  
- Responsive layout structure  

## Description
The layout is built using CSS Grid to place major sections of a webpage.  
The header and footer span the full width, while the sidebar and main content are placed side by side.

## Technologies Used
- HTML5  
- CSS3  

## Purpose
This project was built as part of my frontend practice to understand CSS Grid and modern layout design.

---

This layout serves as a base structure for building real-world web pages.
